🔧 DeepSeek API 配置

📖 如何获取 DeepSeek API 密钥：

  1. 访问 DeepSeek 平台
  2. 登录或注册 DeepSeek 账户
  3. 点击左侧菜单中的 "API Keys"
  4. 点击 "Create Key" 按钮
  5. 复制生成的密钥（格式：sk-xxxxxx）
  6. 回到这个页面，点击"显示/隐藏"按钮
  7. 粘贴密钥到输入框中
  8. 点击"保存 API 密钥"按钮

🚀 配置成功后：

  • 访问 教师端主页面
  • 访问 学生端主页面
  • 即可使用 AI 功能（生成授课文案、智能问答等）

⚠️ 注意事项：

  • API 密钥将保存在浏览器的 localStorage 中
  • 每个浏览器需要单独配置一次
  • 请妥善保管您的 API 密钥，不要泄露给他人
  • 清除浏览器数据会删除已保存的密钥
