# MCP 服务模块

`backend/mcp-server/` 提供 MCP 兼容的图谱工具桥接能力。它不是普通用户访问的主入口，而是给支持 MCP 的客户端或调试工具使用。

## 目标

- 暴露图谱查询工具。
- 复用本项目本地知识图谱。
- 为外部工具提供结构化上下文。

## 常见工具类型

具体工具名以实现为准，通常包括：

- 查询节点。
- 查询关系。
- 搜索概念。
- 获取学习路径。
- 获取某个章节的相关知识点。

## 安全边界

MCP 服务不应：

- 保存 API Key。
- 暴露 `.env`。
- 返回本地个人路径。
- 返回未公开论文或本地参考文件。

## 维护要求

新增或修改 MCP 工具时，需要记录：

- 工具名。
- 输入参数。
- 输出结构。
- 失败返回。
- 是否读取本地数据库。

如果工具输出包含公式或 Markdown，应保持和前端渲染链路兼容。
